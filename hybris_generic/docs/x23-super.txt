Notes
-----
- Full A/B everywhere except shared partitions (userdata, misc, persist, nv*).
- VBMeta is per-slot and per-stage (vbmeta + vbmeta_system + vbmeta_vendor).
  For unsigned OHOS images we set verifiedbootstate=orange, veritymode=disabled
  via fastboot --disable-verity --disable-verification on the slot _b flash.
- nvram, nvdata, nvcfg, protect1, protect2 carry IMEI / radio calibration —
  do NOT touch in N1; preserve as-is. Telephony out of scope for Milestone 4.
- Kernel cmdline already carries firmware_class.path=/vendor/firmware,
  ramoops.* (see /proc/cmdline), and hardware=x23 — no changes needed for N1.
